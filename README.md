## About this APP

This is a very simple Single Page application built using Laravel 7 and Vue.JS 2.6. The full CRUD has been applied.

## Used Technologies
- Laravel 7
- Vue.JS 2.6
- Bootstrap 4.4
- Axios
- SweelAlert 2
- jQuery (Not mandatory to run the app)
